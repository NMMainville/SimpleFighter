OVERVIEW
--------
This game was written in a day to use as an example for people who expressed an interest in designing games. The game was displayed on a television, and controllers were available for people to use. The game restarts once all enemies are dead.

The game can be played online at this link: http://files.myfrogbag.com/bli6du/Clubsfest2.swf

CONTROLS
--------
Arrow Keys:	Move
X:		Jump
Z:		Attack



Built with Flixel 2.43.
